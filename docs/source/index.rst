SunFounder Vincent Kit για το Arduino
======================================

Αυτό είναι ένα κιτ κατάλληλο για αρχάριους και γνώστες του Arduino. Το κιτ, το οποίο χρησιμοποιεί το Mega2560 ως κύρια πλακέτα ελέγχου και περιλαμβάνει επιπλέον 64 ευρέως χρησιμοποιούμενα στοιχεία εισόδου/εξόδου, μονάδες και πολλά άλλα βασικά ηλεκτρονικά στοιχεία
(όπως αντιστάσεις, πυκνωτές, τρανζίστορ κ.λ.π.), βοηθάει σημαντικά στο να υλοποιήσετε και να ολοκληρώσετε επιτυχ τα δικά σας έργα προγραμματισμού. 

Κατεβάστε τον σχετικό κώδικα από τον παρακάτω σύνδεσμο.

* :download:`SunFounder Vincent Kit for Arduino <https://github.com/sunfounder/sunfounder_vincent_kit_for_arduino/archive/refs/heads/master.zip>`

* Ή κάντε "check out" τον κώδικα στο `Vincent Kit for Arduino - GitHub <https://github.com/sunfounder/sunfounder_vincent_kit_for_arduino>`_

Εάν θέλετε να μάθετε άλλα έργα, τα οποία δεν διαθέτουμε, στείλτε παρακαλώ ελεύθερα Email και θα ενημερώσουμε τα, σε απευθείας σύνδεση, τα εγχειρίδιά μας όσο πιο σύντομα γίνεται. Οι προτάσεις σας είναι καλοδεχούμενες.

Το Email επικοινωνίας είναι το: cs@sunfounder.com.

Σημείωση Πνευματικών Δικαιωμάτων
----------------------

Όλο το περιεχόμενο, συμπεριλαμβανομένων μεταξύ άλλων των κειμένων, των εικόνων και του κώδικα σε αυτό το εγχειρίδιο, ανήκει στην Εταιρεία SunFounder. Θα πρέπει να το χρησιμοποιείτε μόνο για προσωπική μελέτη, έρευνα, διασκέδαση ή άλλους μη εμπορικούς ή μη κερδοσκοπικούς σκοπούς, σύμφωνα με τους σχετικούς κανονισμούς και τους νόμους περί πνευματικών δικαιωμάτων, χωρίς να παραβιάζονται τα νόμιμα δικαιώματα του δημιουργού και των σχετικών κατόχων δικαιωμάτων. Για κάθε άτομο ή οργανισμό που τα χρησιμοποιεί για εμπορικό σκοπό χωρίς άδεια, η Εταιρεία διατηρεί το δικαίωμα να προβεί σε κάθε νόμιμη ενέργεια.
