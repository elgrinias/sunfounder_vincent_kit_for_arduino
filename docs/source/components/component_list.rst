Λίστα Εξοπλισμού και Εισαγωγή

Αφού ανοίξετε το πακέτο, παρακαλώ ελέγξτε  εάν η ποσότητα των εξαρτημάτων αντιστοιχεί στην περιγραφή των προϊόντων και εάν όλος ο εξοπλισμός βρίσκεται σε καλή κατάσταση.
